# ActiveX-Barcode-Print
一个用C# 开发的IE浏览器activex 打印插件

## 关于使用

1.在电脑D盘根目录下创建nykjprint文件夹将PrintSet.xml 放入文件夹内并修改需要调用的打印机名称

2.生成安装step.exe 

3.前端调用
example：
<object id="CardAccessor"
            classid="clsid:073A987E-2A7C-4874-8BEE-321E04F4E84E"
            width="200"
            height="200">
</object>
